CUM SE PORNEȘTE PROIECTUL:
1. Instalează PostgreSQL și creează o bază numită 'asociatie_db'.
2. Rulează codul SQL din 'baza_de_date.sql' ca să ai tabelele.
3. Deschide terminalul în folder și scrie: pip install -r requirements.txt
4. Rulează aplicația: python app.py
5. Intră în browser la: http://127.0.0.1:5000
